Java基础知识-集合(1)

作者：Hys 更新于

1 7月 2020

前言：

java中集合主要是两大接口派生来的，一类是collection，另一个是map。collection主要是list，set，queue三个子接口。

collection下的集合：list，set，queue

1.list:

list下分为三种，ArrayList、Vector以及LinkedList。ArrayList和Vector底层数据结构是数组，LinkedList底层数据结构是双向链表（JDK1.6之前为双向循环链表）。

区别：

ArrayList与Vector的区别主要是线程安不安全。Vector是安全的，ArrayList不安全，且ArrayList使用于频繁查找。
ArrayList于LinkedList特点与区别：

1.两者都是线程不安全的。

2.ArrayList底层是数组，LinkedList底层是双向链表

3.ArrayList查找，增加元素时间复杂度为O(1),删除复杂度为O(n)。LinkedList头尾插入删除时间复杂度近似O(1)，其他位置的插入删除以及查找的时间复杂度为O(n)

4.ArrayList能快速随机访问（通过下标）

5.ArrayList 的空 间浪费主要体现在在 list 列表的结尾会预留一定的容量空间，而 LinkedList 的空间花费则体现在它的每一个元素都需要消耗比 ArrayList 更多的空间（因为要存放直接后继和直接前驱以及数据）。

重点：ArrayList扩容机制
int newCapacity = oldCapacity + (oldCapacity >> 1),所以 ArrayList 每次扩容之后容量都会变为原来的 1.5 倍左右（oldCapacity 为偶数就是 1.5 倍，否则是 1.5 倍左右）！ 奇偶不同，比如 ：10+10/2 = 15, 33+33/2=49。如果是奇数的话会丢掉小数。

具体请见：ArrayList-guide哥

2.Set:

Set分为HashSet、LinkedHashSet 和 TreeSet，主要的特点就是非重复性，Set中的元素都能保证每个元素都是不重复的。主要区别在于底层数据结构。 HashSet：哈希表（基于Hashmap实现）。LinkedHashSet：链表和哈希表，元素的插入和取出顺序满足先进先出。TreSet：红黑树，元素是有序的。
