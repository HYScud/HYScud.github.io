1.基本数据类型

Java中有 8 种基本数据类型，分别为：byte，int，short，long，char，boolean，float，double。其中这八种数据类型的默认值以及空间如下：
基本数据类型 位数 字节 默认值 取值范围
byte 8 1 0 -128~127
int 32 4 0 -2147483648 ~ 2147483647
short 16 2 0 -32768 ~ 32767
long 64 8 0L -9223372036854775808 ~ 9223372036854775807
char 16 2 'u0000' 0 ~ 65535
float 32 4 0f 1.4E-45 ~ 3.4028235E38
double 64 8 0d 4.9E-324 ~ 1.7976931348623157E308
boolean 1 fasle true、fasle

注意：

1.Java 里使用 long 类型的数据一定要在数值后面加上 L，否则将作为整型解析

2.char a = 'h'char :单引号，String a = "hello" :双引号。

2.包装类型

八中基本类型对应的包装类型Integer、Byte、Short、Long、Boolean、Character、Double、Float。 其中Byte,Short,Integer,Long 这 4 种包装类默认创建了数值 [-128，127] 的相应类型的缓存数据，Character 创建了数值在 [0,127] 范围的缓存数据，Boolean 直接返回 True or False。 也就是只要取值在对应的范围，就是去常量池取值，当超出范围时，才会创建新的对象。例如：Integer a=2，Integer b=2。a==b是true。而a=128，b=128。这时候a==b是fasle。
